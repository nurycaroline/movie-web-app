# Movie Web App

## Comands

Instalar

```
-> yarn
```

Executar localmente

```
-> yarn start
```

Gerar build

```
-> yarn build
```

Executar teste em modo watch

```
-> yarn test:watch
```

Executar testes
```
-> yarn test
```

Executar localmente a documentacao dos componentes

```
-> yarn docz
```

Gerar build da documentacao

```
-> yarn docz:build
```
